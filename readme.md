# Интеллектуальная система распознавания речи для голосового управления устройствами
## Описание системы
Система предназначена для сбора, обработки и анализа речевых данных, полученных через голосовые команды в режиме реального времени, с целью управления устройствами без непосредственного контакта с ними, обеспечения удобства взаимодействия с устройствами.

---

## Возможности системы

- Авторизоваться пользователям с разными ролями (администратор, пользователь, специалист)

- Управлять устройствами умного дома (добавлять, редактировать, удалять)

- Использовать голосовые команды для взаимодействия с системой

- Анализировать данные с помощью различных алгоритмов (машинное обучение, статистический анализ)

- Формировать ответы и решения на основе полученных данных

---

## Используемые технологии
### Языки программирования
- Python 3.12 - основной язык разработки

### Основные библиотеки:
- Tkinter - для создания графического интерфейса

- SpeechRecognition - для распознавания речи через Google Speech API

- Threading - для многопоточности и фоновых задач

- JSON - для хранения данных пользователей и устройств

- Dataclasses - для создания моделей данных

### Архитектурные паттерны:
- MVC (Model-View-Controller) - основная архитектура

- Стратегия (Strategy) - для алгоритмов анализа

- Команда (Command) - для операций с отменой

- Фабрика (Factory) - для создания компонентов

## Запуск программы
### Шаг 1. Клонирование репозитория
```
git clone https://github.com/aChainsmoker/OMIS-LR-6.git
```

### Шаг 2: Запуск приложения
```
cd OMIS-LW-6
python3 main.py
```
### Шаг 3: Авторизация в системе
После запуска используйте следующие тестовые данные для входа:

```
Логин: admin, Пароль: admin123 (администратор)

Логин: user1, Пароль: user123 (обычный пользователь)

Логин: specialist, Пароль: spec123 (специалист)
```
Или создайте свой профиль, нажав на кнопку `Регистрация`

### Шаг 4: Использование системы
После авторизации вы увидите главное меню

Для использования голосового управления нажмите "Голосовой ввод"

Для управления устройствами перейдите в раздел "Устройства"
